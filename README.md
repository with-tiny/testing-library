# tiny-testing-library

Tiny testing library for little projects

## Installation

```
npm install -D tiny-testing-library
```

## Usage

Set package.json test script to tiny-testing-library

```
{
  "scripts": {
    "test": "tiny-testing-library"
  }
}
```

And just run

```
npm run test
```
